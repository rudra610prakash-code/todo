# ✅ Todo App

A clean, simple, and elegant to-do list application to organize your daily tasks efficiently.

## 🎯 Features

- ✨ **Add Tasks** - Quickly add new tasks to your list
- ✏️ **Edit Tasks** - Modify existing tasks with ease
- 🗑️ **Delete Tasks** - Remove completed or unwanted tasks
- 💾 **Local Storage** - Your tasks are saved automatically in your browser
- 📱 **Responsive Design** - Works seamlessly on all devices
- 🎨 **Clean UI** - Minimalist and user-friendly interface

## 🚀 Quick Start

1. Open `index.html` in your web browser
2. Type your task in the input field
3. Click **Add** to create a task
4. Use **Edit** to modify or **Remove** to delete tasks
5. Your tasks are automatically saved!

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox
- **JavaScript** - Dynamic task management
- **Local Storage API** - Persistent data storage

## 💡 How It Works

- Tasks are stored in your browser's **localStorage**
- Each task comes with **Edit** and **Remove** options
- Changes are instantly reflected and saved
- Close and reopen the app - your tasks will still be there!
